## About Nyaho Test Project

This project is developed with laravel 9.45.1 

Clone this project and run the the following commands in the root folder of the project to set up.

`composer install` to install the laravel dependencies

`npm install` (this will help istall vite which is used for the frontend asset bundling)

`npm run dev`  or `npm run build` once

Also create .env file and copy the contents of .env.example into it (to be able to connect to the DB)
